# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup with Nx monorepo
- BHP Calculator library with core calculation engine
- Chart Components library with ECharts integration
- Data Generator library for synthetic data
- Demo application showcasing all libraries
- CI/CD workflows with GitHub Actions
- Automated deployment to GitHub Pages
- Comprehensive documentation
- Unit tests with Vitest
- ESLint configuration
- Prettier formatting
- Dependabot configuration
- GitHub issue and PR templates

### Features
- Real-time BHP calculations
- Interactive charts (line, gauge, scatter, bar)
- Theme switching (light/dark mode)
- Simulation controls (start/stop/reset)
- Responsive design
- Material Design UI
