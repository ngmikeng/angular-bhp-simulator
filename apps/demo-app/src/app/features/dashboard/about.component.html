BHP Real-Time Simulator

A demonstration of bottomhole proppant concentration calculation from surface measurements using Angular and real-time data streaming

About the Project

The BHP Real-Time Simulator is an educational application that demonstrates how bottomhole proppant concentration (BHP) is calculated from surface measurements during hydraulic fracturing operations. The application simulates real-time data streaming and applies a backward-looking incremental algorithm to compute BHP values.

This project showcases modern web development practices including reactive programming with RxJS, component-based architecture with Angular, and real-time data visualization with ECharts.

The BHP Algorithm

The backward-looking incremental BHP algorithm calculates the bottomhole proppant concentration based on:

  • Slurry Rate: The pumping rate at surface (barrels per minute)
  • Treating Pressure: The pressure measured at surface (PSI)
  • Proppant Concentration: The concentration of proppant in the slurry (pounds per gallon)

The algorithm maintains a sliding time window of historical data and applies fluid mechanics principles to estimate the proppant concentration at the bottom of the wellbore, accounting for factors such as:

  • Fluid friction in the wellbore
  • Hydrostatic pressure from fluid column
  • Proppant settling and transport
  • Time delays between surface and downhole
